# Wool & Wonder Studio Website

Single-page marketing site for a felting class business.

## Quick Start

1. Open `index.html` in a browser.
2. Edit text directly in `index.html`.
3. Adjust colors and layout tokens in `styles.css`.

## Files

- `index.html`: page structure + content.
- `styles.css`: visual system, responsive layout, animations.
- `app.js`: staggered reveal-on-scroll behavior.

## Customize Fast

- Instructor quote: `index.html` section `#about`.
- Booking email: `index.html` section `#book`.
- Business name and city: header/footer in `index.html`.
